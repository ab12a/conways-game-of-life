# Conway's Game of Life

## How to Run

```
pip install pygame
python life.py --width 60 --height 30 --fps 10
```

## Controls
- `Space` – Play / Pause
- `N` – Single Step (when paused)
- `R` – Random Fill
- `C` – Clear Grid
- `S` – Save Pattern
- `L` – Load Pattern
- `Mouse Click` – Toggle cell (when paused)

## Pattern File Format
```
# Pattern: Glider
1,0
2,1
0,2
1,2
2,2
```
